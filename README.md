# Medical-Teaching-Coordination

Final project for the OOP subject.

## Description

Medical Teaching Coordination example project with academic intended use only.

## Getting Started

### Dependencies

* Install chocolatey if not present (Windows): 
https://chocolatey.org/install#individual
* Install maven if not present (powershell as admin):
```
choco install maven 
```
* Install make if not present (powershell as admin):
```
choco install make 
```
### Installing

* Clone this repository using Git.

### Executing program (Windows)

* Open your console at the root folder of the cloned repository.
* Change directory to MedicalTeachingCoordination:
```
cd MedicalTeachingCoordination
```
* The makefile contains all the commands needed, use the following command to show them:
```
make help
```
